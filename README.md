- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma
- SQLite (for demo purposes)

## Development

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
